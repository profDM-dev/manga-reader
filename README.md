# YoruManga - Project Complete

Live Demo: https:[Launch Yorumanga](//profdm-dev.github.io/manga-reader/Manga_reader/)

## What Has Been Built

A fully functional manga-reading website with Japanese-inspired design.

### Deliverables

5 HTML Pages:
- index.html - Home/Browse with infinite scroll
- manga.html - Manga details & chapters  
- reader.html - Vertical scroll reader
- login.html - Authentication
- library.html - Bookmarks & history

7 JavaScript Modules:
- api.js - MangaDex API integration
- auth.js - User authentication
- ui.js - UI helpers & dark mode
- home.js - Home page logic
- manga.js - Details page logic
- reader.js - Reader logic
- library.js - Library logic

## Features Implemented

Core Features:
- Infinite scroll manga listing
- Search by title (debounced)
- Filter by status & sort order
- Manga details with full info
- English chapter list (sorted)
- Vertical scroll reader
- Login/Register system
- Bookmarks functionality
- Reading history tracking
- Dark mode with persistence
- Fully responsive design

Technical Stack:
- Pure vanilla JavaScript (no frameworks)
- Tailwind CSS via CDN (no build)
- MangaDex API integration
- localStorage for user data
- In-memory caching
- Lazy image loading

## Tech Stack
HTML5 • CSS3 (Tailwind CDN) • Vanilla JavaScript • MangaDex API
